## 🔍 Troubleshooting & Common Issues

### ❌ Script doesn't execute / nothing happens

**Cause:** Executor not properly attached to the game process.
**Fix:** Close and reopen your executor. Make sure the popular title is fully loaded before injecting.

### ❌ Auto Green fires too early or too late

**Cause:** `delay_ms` is set too low or too high for your current ping.
**Fix:**
```bash
# Increase delay for high-ping connections
"delay_ms": 25

# Decrease for low-ping (LAN/fiber)
"delay_ms": 8
```

### ❌ Ball MAG not activating

**Cause:** `pull_radius` is below the threshold for current server physics settings.
**Fix:** Increase `pull_radius` to `10.0` and restart the session.

### ❌ Script crashes on Android after 10 minutes

**Cause:** Device entering power-saving mode, killing background processes.
**Fix:** Go to **Battery Settings** and whitelist the script app from optimization.

### ❌ iOS certificate revoked

**Cause:** Developer certificate expired.
**Fix:** Re-download the latest `.ipa` from the release page. Certificates are refreshed monthly.

### ❌ "Config not found" error on launch

**Cause:** First-time run or corrupted config file.
**Fix:**
```bash

rm ~/Library/Application\ Support/BLScript/config.json
# Relaunch the script — it will auto-generate a fresh config
```

### ❌ HUD not visible in-game

**Cause:** HUD position set outside screen bounds, or opacity set to 0.
**Fix:** Reset HUD settings:
